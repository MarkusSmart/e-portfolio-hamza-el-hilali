## Live Demo
[View Portfolio](https://markussmart.github.io/e-portfolio-hamza-el-hilali/) (Update this link after deploying to GitHub Pages)

## Local Development
To view locally:
1. Clone the repository
2. Open `index.html` in your web browser

## Deployment to GitHub Pages
1. Create a new repository on GitHub named `e-portfolio-hamza-el-hilali`
2. Push this code to your repository:
   ```
   git remote add origin https://github.com/YOUR_USERNAME/e-portfolio-hamza-el-hilali.git
   git branch -M main
   git push -u origin main
   ```
3. Go to repository Settings > Pages
4. Under Source, select "Deploy from a branch"
5. Select "main" branch and "/ (root)" folder
6. Save and wait for deployment (usually 1-2 minutes)
7. Your site will be available at `https://YOUR_USERNAME.github.io/e-portfolio-hamza-el-hilali/`
